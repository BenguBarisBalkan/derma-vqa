Visual Question Answering in Dermatology

This project is about Visual Question Answering (VQA) in dermatology field. Although VQA is used in medical field, especially in X-Ray studies it's use in deramtology fields is limited due to data limitations. We aimed to create a dermatology VQA dataset from scratch and use it to train a VQA pipeline. The pipeline includes an image classifier and a large language model. The image classifier is used to classify the skin condition in the image and the large language model is used to answer the question based on the classified skin condition.
